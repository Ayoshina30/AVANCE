# AVANCE - Fitness & Music Tracking Application

A desktop application that combines workout tracking with music recommendations using the Last.fm API.

## Features
- Custom dark-themed UI with sidebar navigation
- Real-time artist recommendations based on workout history
- Performance analytics with data visualization
- User onboarding and accessibility features

## Technologies
Python • CustomTkinter • pandas • matplotlib • Last.fm API • Figma

## Installation
1. Clone the repository
2. Install dependencies: `pip install customtkinter pandas matplotlib requests`
3. Add your Last.fm API key to `config.py`
4. Run: `python main.py`
